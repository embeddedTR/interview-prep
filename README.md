# embeddedTR — Mülakat Hazırlık Rehberi

Türkiye'deki embedded software mülakat süreçlerine hazırlık için hazırlanmış soru ve cevap bankası.

## Bölümler

| Bölüm | Konu |
|---|---|
| [C Programlama](./c-programlama/) | Pointer, volatile, bitwise, bellek yönetimi |
| [Donanım](./donanim/) | I2C, SPI, UART, Interrupt, DMA |
| [RTOS](./rtos/) | Görev yönetimi, senkronizasyon, deadlock |
| [Kod Egzersizleri](./kod-egzersizleri/) | Circular buffer, debounce, driver yazımı |

## Zorluk Seviyeleri

- 🟢 Giriş seviyesi
- 🟡 Orta seviye
- 🔴 İleri seviye

## Katkıda Bulun

Yeni soru eklemek, cevap düzeltmek veya kendi mülakat deneyimini paylaşmak istersen PR aç.
